## Abgabeumfang (03.07.2026)

Zum Abgabetermin müssen folgende Artefakte vollständig vorliegen:

- [x] Dokumentation (Entwurf, Quellcode-Doku, Testfälle, Nutzerhandbuch)
- [ ] Quellcode mit Autorenkommentaren je Klasse/Methode
- [ ] Testfälle, Testdaten und Systemtestergebnisse
- [ ] Nutzerhandbuch / Quickguide für Endanwender
- [ ] Tabellarische Autorenübersicht in der Einleitung

!!! info "Autoren-Kennzeichnung"
    Im Quellcode **muss** erkennbar sein, welche(r) Programmierer:in für die jeweilige
    Klasse/Methode verantwortlich ist. Im Dokument gilt dasselbe: jeder Abschnitt
    trägt eine Autorenangabe.
